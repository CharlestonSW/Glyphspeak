About Jarvis Prompts

This folder contains seed and helper prompts used to initiate or steer Jarvis sessions. These prompts pair with corresponding scrolls in scroll/ (e.g., scroll_primacy.yaml) to establish operating modes and continuity anchors.

Contents

All files and folders in this directory are listed below.

  • start_with_scroll_primacy.txt
    Seed prompt for launching Jarvis with Scroll Primacy; best used alongside scroll_primacy.yaml.
  • start_with_scroll_primacy_via_url.txt
    Variant of the seed prompt that references scrolls by full site URL (useful when file-path context is unavailable).
